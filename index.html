Zelda‑like MVP

Rectangles only. Keyboard + Gamepad. Includes: movement, sword, enemies, hearts, rupees, chest, sign, push blocks, doors (normal/locked/clear/switch‑only), grass, cracked walls, pots, bombs, boomerang, bow, switches.

  • Move: WASD / Arrows / Left‑stick / D‑pad (Hold Shift / LT/RT for sprint)
  • Attack/Use: J / Space / Gamepad A (0)
  • Interact/Pick up: K / E / Gamepad X/Y/B (2/3/1)
  • Cycle item: Q / Gamepad RB/LB (5/4)
  • Pause: Esc / Gamepad Start (9)
  • Fullscreen: F
  • Pad debug overlay: ;

Tip: connect a controller before pressing Start.
